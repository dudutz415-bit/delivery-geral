Como cadastrar lojas e produtos no Delivery Geral via Firebase

1 Acessar o Firebase Console
  1. Abra seu navegador e acesse: console.firebase.google.com
  2. Faca login com sua conta Google
  3. Clique no projeto "delivery-geral-af64e"
2 Abrir o Firestore Database
  1. No menu lateral esquerdo, clique em "Firestore Database"
  2. Clique em "Iniciar colecao" ou "+ Iniciar colecao"
  3. Ou, se ja houver colecoes, clique em "+ Adicionar colecao"
Dica: O Firestore e o banco de dados em tempo real. Tudo que voce cadastrar aparece instantaneamente no app!
3 Criar a colecao "stores" (Lojas)
  1. Clique em "+ Iniciar colecao"
  2. Digite o ID da colecao: stores
  3. Clique em "Proximo"
  4. Na tela de adicionar documento, selecione "Gerar ID automatico" (ou digite um ID unico, ex: mercado01)

Campos obrigatorios da loja:

Campo Tipo Obrigatorio Descricao
name string Sim Nome da loja (ex: Supermercado Central)
niche string Sim Categoria da loja. Veja tabela abaixo
email string Sim Email do lojista para login
ownerId string Sim UID do usuario Firebase do lojista
products array Nao Lista de produtos (veja passo 5)
emoji string Nao Emoji que representa a loja (ex: 🛒)
bg string Nao Cor de fundo HEX (ex: #FFF3ED)
rating number Nao Nota de avaliacao (ex: 4.5)
deliveryTime string Nao Tempo medio de entrega (ex: 30-45 min)

4 Exemplo completo de uma loja

Ao criar um documento na colecao stores, preencha os campos assim:

{ "name": "Supermercado Central", "niche": "supermarket", "email": "mercado@exemplo.com", "ownerId": "uid-do-usuario-firebase", "emoji": "🛒", "bg": "#FFF3ED", "rating": 4.6, "deliveryTime": "30-45 min" }
Importante: O campo ownerId deve ser o UID do usuario que criou a conta de lojista no app. Para obter o UID, o lojista deve fazer login pelo painel de lojista (lojista.html) uma vez. Voce encontra o UID no campo "UID do usuario registrado".
5 Cadastrar produtos dentro da loja
  1. Abra o documento da loja que voce acabou de criar na colecao stores
  2. Clique no botao "+" para adicionar um novo campo
  3. Digite o nome do campo: products
  4. Selecione o tipo: array
  5. Clique em "Adicionar item" para cada produto

Estrutura de cada produto no array:

Campo Tipo Obrigatorio Exemplo
id string Sim "prod-001"
name string Sim "Arroz Branco"
price number Sim 5.49
desc string Sim "Arroz branco tipo 1"
emoji string Nao "🍚"
popular boolean Nao true ou false

Exemplo de produto (no Firestore, adicione como map dentro do array):

{ "id": "arroz-001", "name": "Arroz Branco T1", "price": 5.49, "desc": "Arroz branco tipo 1, 1kg", "emoji": "🍚", "popular": true }
6 Verificar no app
  1. Salve tudo no Firestore (clique em "Salvar")
  2. Abra o app no navegador: delivery-geral.netlify.app
  3. A loja e os produtos aparecerao automaticamente na home!
Rapido: O Firestore atualiza em tempo real. Voce nao precisa recarregar a pagina - os dados aparecem sozinhos!
7 Criar conta de lojista no app
  1. Abra o arquivo lojista.html no navegador
  2. Clique em "Cadastrar loja" e preencha os dados
  3. O sistema cria automaticamente:
    • Um novo usuario no Firebase Authentication
    • Um documento na colecao stores
  4. O lojista pode usar o email e senha cadastrados para fazer login no painel
Atenção: O cadastro pelo app cria um ownerId automaticamente. O lojista nao precisa saber esse ID - o sistema gerencia tudo.
8 Criar conta de entregador
  1. Abra o arquivo entregador.html no navegador
  2. Preencha os dados e escolha o tipo de veiculo:
    • 🏍️ Moto
    • 🚗 Carro
    • 🚚 Caminhão
    • 🚲 Bicicleta
  3. Cadastre e faca login
  4. Os pedidos aparecerao automaticamente quando o admin atribuir a entrega
